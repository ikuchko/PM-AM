## Setup

Clone this repository:
```

```

Open terminal and run Postgres:
```
$ postgres
```

Open a new tab in terminal and create the `hair_salon` database:
```
$ psql
$ CREATE DATABASE ;
$ psql
```

Navigate back to the directory where this repository has been cloned and run gradle:
```
$ gradle run
```
